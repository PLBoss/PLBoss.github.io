Css

简介

CSS被称为层叠样式单，用与描述结构文档的表现方式，主要用于网页风格的设计，包括字体大小的设计，颜色，背景以及元素的精确定位，增强文档的表现力。

基本结构

p {color:red ;size:20px;} 设置p标签的颜色为红色，字体大小为20px

使用css样式的方法

内嵌样式

内部样式就是直接在标签内设置该标签的样式属性

<p style="color:red ,frontsize:20px;front-family:微软雅黑> 内部样式设置</p>

内部样式

也可以将样式属性直接放在头部的style标签中

<head>
<style>  
        p{
            color:red;
            font-size:20px;
            
            }
         </style>
</head>

外部样式

外部样式是直接将所有的样式写进一个css文件中，然后从HTML文件中直接调用该文件
头部标签中加入 

<link rel="stylesheet" href="style.css"> #这里注意将css文件和HTML文件放在同一级目录

事件

HTML DOM 事件 (w3school.com.cn)

要想知道什么是事件监听，首先先聊聊什么是事件？ HTML 事件是发生在 HTML 元素上的“事情”。比如：页面上的 按钮被点击 、 鼠标移动到元素之上 、 按下键盘按键 等都是事件。 事件监听是JavaScript 可以在事件被侦测到时执行一段逻辑代码。

常用事件

mousemove 当指针在元素上方移动时，发生此事件。
mouseout 当指针移动到元素或其中的子元素上时，发生此事件。
focus 获取焦点的事件
focusin 在元素即将获得焦点时发生此事件
