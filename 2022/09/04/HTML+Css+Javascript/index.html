Css

简介

CSS被称为层叠样式单，用与描述结构文档的表现方式，主要用于网页风格的设计，包括字体大小的设计，颜色，背景以及元素的精确定位，增强文档的表现力。

基本结构

p {color:red ;size:20px;} 设置p标签的颜色为红色，字体大小为20px

使用css样式的方法

内部样式

内部样式就是直接在标签内设置该标签的样式属性

<p style="color:red ,frontsize:20px;front-family:微软雅黑> 内部样式设置</p>

也可以将样式属性直接放在头部的style标签中

<head>
<style>  
        p{
            color:red;
            font-size:20px;
            
            }
         </style>
</head>

外部样式

外部样式是直接将所有的样式写进一个css文件中，然后从HTML文件中直接调用该文件
头部标签中加入 

<link rel="stylesheet" href="style.css"> #这里注意将css文件和HTML文件放在同一级目录

导入外部样式表(style标签中加入）
@import “style.css”;

css的特性

层叠性

权重低的选择器可以被权重高的选择器覆盖掉
#a ID选择器 权重向下依次递减
.b 类选择器
P 标签选择器

继承性

子标签的属性可以有父标签的属性定义
有其父必有其子吗？答案是不一定的，比如a标签不能继承颜色，h标签不能继承字体大小

优先性

!important>行内样式>ID选择器>类选择器>标签选择器>通配符>继承>浏览器属性

css样式选择器

h1{} //标签选择器
*{}  //通配符
#id{} //id选择器
.class{} //类选择器
p,div{}  //群选择器
p .class a {} //包含选择器 选择某个特定标签下的子标签
ul.class >li>a {} //子类选择器
ul h2~p //兄弟选择器

动态伪类选择器

伪类选择器并不是正真的选择元素的选择器，而是针对伪元素

li:bedfore{ } //在li标签之前插入
li:after { } //在li标签后插入
li:before { content:counter(order,upper-roman)".";} //设置编号

css选择器

结构性伪类选择器，利用文档结构树来实现元素的过滤，类似于正则表达式的匹配，这个只不过匹配的是文档元素

：root //匹配文档的根元素
:first-child //匹配父元素的第一个子元素
:last-child  // 匹配父元素的最后一个子元素
UI伪类选择器

只有存在某种状态才可能某种样式出现
：hover //鼠标悬浮的样式
:active //按住左键不松开的样式
:focus //获得指针输入时的样式

单选框与复选框样式
input[type="checkbox"]：checked{}  //单选框
input[type="text"]::selection{}    //复选框
属性选择器
[class^=d]匹配类名以d开始的每个元素
[class$=d]匹配以类名d结尾的每个元素
